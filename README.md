# ミテルンデス (Miterundesu) 🔍

**保存できない拡大カメラアプリ** — 美術館・博物館・劇場で安心して使える、プライバシー重視の拡大鏡

[![Swift](https://img.shields.io/badge/Swift-5.9-orange)](https://swift.org)
[![iOS](https://img.shields.io/badge/iOS-17.0+-blue)](https://developer.apple.com/ios/)

## 概要

ミテルンデスは、視覚障害者や高齢者が**文化施設で安心して使える拡大カメラアプリ**です。撮影した画像は10分後に自動削除され、スクリーンショットや画面録画も防止。施設側も来場者も安心できる仕組みを実現しています。

## ✨ 主な機能

- 📷 **高倍率ズーム** — ピンチ＆ボタンで最大200倍まで拡大
- ⏱ **10分自動削除** — 撮影画像は10分後に自動で消去
- 🛡 **スクリーンショット・画面録画防止** — 不正コピーをブロック
- 🎭 **シアターモード** — 美術館・劇場向けの静音・暗画面モード
- 📰 **プレスモード** — 報道関係者向け認証付きモード
- 🌊 **透かし（ウォーターマーク）** — 保存画像にウォーターマーク付与
- 🌐 **日本語 / 英語対応**
- ♿ **VoiceOver完全対応** — アクセシビリティファースト設計

## 🛠 技術スタック

- **言語:** Swift
- **UI:** SwiftUI
- **カメラ:** AVFoundation
- **データ:** Core Data
- **バックエンド:** Supabase (認証)
- **対応OS:** iOS 17.0+

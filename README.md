# 🟢 ミテルンデス（miterundesu）

## 📱 コンセプト
**「見る」だけの撮影できないカメラアプリ。**  
保存も共有もできない、**10分だけ写真を撮って見れる。**  
弱視や老眼で悩む人に役立つアプリ。

---

## 🎨 デザインテーマ
| 要素 | 内容 |
|------|------|
| **メインカラー** | #538234（特徴的な深緑） |
| **アクセントカラー（シアターモード時）** | #E97B2A（落ち着いたオレンジ） |
| **トーン** | シンプル・落ち着き・非保存性を感じる透明感 |
| **フォント** | San Francisco（iOS標準）をベースに可読性重視 |
| **ロゴ** | 「ミテルンデス」ロゴを上部中央に配置 |

## 🧭 画面構成

### 🌿 メイン画面
- 上部中央：「**ミテルンデス**」ロゴ  
  - その下に、**テキスト「画像は保存できません。」が左→右に無限スクロール**  
  - さらにその下に、「📘説明を見る」ボタンを配置  
- 上部左：「**シアターモード トグル**」  
  → ON/OFF で切り替え（トグルスイッチ形式）  
  → 有効時は**撮影・一時停止・設定操作がすべて無効化される**  
  → 画面全体のテーマカラーが**#E97B2A（オレンジ）**に切り替わる  
  → プレビュー画面は**拡大表示専用**となり、UIボタンがフェードアウト  
  → 約15秒間操作がない場合、UI全体が自動で非表示（再タップで復帰）  
- 上部右：「**設定（⚙）**」アイコン  
  → 言語設定、最大拡大率の変更など
- 中央：**カメラプレビュー（ライブ映像）**  
  - 拡大・縮小可能（ピンチ操作対応）  
  - **デフォルト最大拡大率：×100**  
  - スクリーンショット完全無効化
- 下部中央：「**シャッターボタン**」（一時停止機能付き）  
  → 撮影後は10分間だけアプリ内でプレビュー可能（その後自動削除）
- 下部左：「**最近撮影した画像サムネイル**」  
  → タップで10分以内の画像を閲覧（残り時間表示付き）
- 下部右：「**倍率表示**」（例：×1.5、×100）

## 🎬 シアターモード仕様
| 項目 | 内容 |
|------|------|
| **操作方法** | 上部左のトグルスイッチをON/OFFで切り替え |
| **ON状態のカラー** | #E97B2A（オレンジトーンに切り替え） |
| **撮影操作** | 無効化し、シャッターボタンは半透明で「撮影不可」と表示 |
| **UI挙動** | 撮影系UIを非表示、拡大表示UIのみ残す |
| **拡大操作** | ピンチ操作によるズームイン・アウトのみ有効 |
| **用途** | 映画館・美術館・博物館など、静かな場所で対象を拡大して見る時に使用 |
| **OFF時の動作** | 通常モードへ即時復帰（緑トーンに戻り、操作再有効化） |
| **説明画面** | シアターモード時には、環境に合わせて別の説明文を表示（目的は同じ） |

## 🧩 特徴まとめ
- ❌ **保存不可**：どの操作でもデータは残らない  
- 🔒 **共有不可**：他アプリへの転送禁止  
- 📸 **スクリーンショット無効化**  
- ⏳ **自動削除**：撮影から10分で画像削除  
- 🎞️ **一時停止可能**：撮影ではなくプレビュー停止  
- 🔍 **最大拡大率×100（設定で変更可能：×10〜×200）**  
- 🎬 **シアターモードトグル搭載**：静音＋撮影禁止＋低反射UI表示  
- 🌈 **色覚・視覚配慮**：高コントラストUI・大きなタップ領域
- ♿ **多様な利用者設計**：弱視・老眼・高齢者に配慮した構成
